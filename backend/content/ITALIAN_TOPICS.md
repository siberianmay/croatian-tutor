## Key Differences from Croatian

For reference when adapting teaching methodology:

| Aspect | Croatian | Italian |
|--------|----------|---------|
| Cases | 7 grammatical cases | No cases (except pronouns) |
| Articles | None | Definite + indefinite articles |
| Verb aspect | Perfective/imperfective pairs | No aspect; tense carries meaning |
| Verb tenses | Fewer tenses | More tenses (esp. subjunctive) |
| Word order | Flexible (case-marked) | SVO (fairly fixed) |
| Noun gender | 3 (masc/fem/neuter) | 2 (masculine/feminine) |
| Subjunctive | Minimal/literary | Active daily use |
| Clitics | Complex ordering rules | Less complex but position-dependent |

## Notes for Implementation

1. **Verb focus:** Italian curriculum is more verb-tense heavy than Croatian's case-heavy approach
2. **Subjunctive mastery:** Critical for Italian from B1 onwards; requires dedicated practice
3. **Article system:** Unique challenge for speakers of article-less languages
4. **Regional variation:** Passato remoto usage varies greatly by region (North rarely uses it in speech)
5. **Formal register:** Lei form requires separate conjugation patterns throughout
